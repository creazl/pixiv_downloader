随机涩图API

“别整天搁那儿爬来爬去，小孩子吗？”
都是p站的图，你怎么不直接爬p站

部分图片可能会出现404，那就是寄了
可能是画师把p站的图删了或者设置了非公开

API需要维护，可能会偶尔出现请求失败的情况

请求过快会暂时封禁ip（正常来讲谁TM一秒看几张涩图，真要看就给我请求JSON）

使用说明

  • 请求方式：GET / POST
  • 请求地址：

https://image.anosu.top/pixiv/direct （直接跳转）

https://image.anosu.top/pixiv/json （返回json）

  • 返回格式：302跳转 / JSON
  • 参数介绍：
名称 类型 可选值
num int [1,15]
r18 int 0, 1, 2,
keyword string 字符串
proxy string 如 i.pixiv.re

参数说明：

  1. num为返回图片的数量，仅请求json时有效
  2. r18为返回图片的分级，0为全年龄，1为18+，2为随机返回一种分级的图片，分别为50%的概率
  3. keyword为图片tag所包含的关键字
  4. proxy图片链接所使用的反代地址，推荐有能力的人自己搭建，网上有教程

调用示例

  • 直接调用

    • https://image.anosu.top/pixiv/direct

  • 关键词查询

    • https://image.anosu.top/pixiv/direct?keyword=心海

  • 返回json

    • https://image.anosu.top/pixiv/json?num=3
